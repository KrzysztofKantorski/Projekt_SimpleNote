# SimpleNote 
Platform for creating and sharing notes between users. Notes can be added with phone camera (google ml kit) or simply editing text. Users can browse, add and coment community notes.
Notes are categorized with subjects and tags. Admin Panel enables only admin user to ban other users, hide comments and browse platform statistics.

Project consists of three main parts
* **Backend** Written in C# .NET
* **Admin Panel** Written in Typescript
* **Mobile App** Written in Flutter
